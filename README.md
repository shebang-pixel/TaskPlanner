# Task Planner

A simple and intuitive Flutter task management application that helps you organize, track, and manage your daily tasks efficiently.

## Features

- Simple and intuitive interface
- Create, update and delete tasks
- Set due dates and reminders
- Organize tasks by categories
- Track progress with statistics
- Diary and gratitude journal

## How The App Works

Task Planner allows users to efficiently manage their tasks and deadlines. Users can:

- Create tasks with titles and descriptions
- Assign due dates and set reminders for important deadlines
- Organize tasks by categories for better management
- Track their progress with detailed statistics
- Maintain a personal diary and practice gratitude journaling
- Edit or delete tasks as needed

The app provides an intuitive interface that makes task management simple and stress-free.

## Installation Instructions

### Prerequisites
- Flutter SDK installed ([Install Flutter](https://flutter.dev/docs/get-started/install))
- Dart SDK (comes with Flutter)
- Android Studio or Xcode (for emulator/device)

### Setup Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/shebang-pixel/TaskPlanner.git
   cd TaskPlanner
   ```

2. Get Flutter dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

   Or run on a specific device:
   ```bash
   flutter run -d <device-id>
   ```
